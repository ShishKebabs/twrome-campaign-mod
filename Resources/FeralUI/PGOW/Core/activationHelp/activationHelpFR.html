Aide & instructions pour l’activation

En quoi consiste l’activation ?

L’activation est une méthode de protection contre la copie qui permet de vérifier que le jeu Feral que vous utilisez est bien authentique.

Pour activer votre jeu, vous aurez besoin d’une Clé Produit qui est uniquement fournie avec les copies authentiques des jeux de Feral.

Une fois que vous avez saisi votre Clé Produit, vous pouvez choisir si vous souhaitez :

  1. Jouer avec le DVD du jeu dans le lecteur DVD de votre Mac (pourvu que vous possédiez la version DVD).
  2. Activer le jeu en ligne Activer le jeu en ligne en envoyant la Clé Produit aux serveurs de Feral.

Vous aurez besoin d’accomplir l’une des deux tâches suivantes avant de pouvoir jouer :

Entrer la Clé Produit

Si vous avez un exemplaire physique du jeu, vous trouverez la Clé Produit au dos du manuel imprimé. Voici ce à quoi elle ressemblera :

Clé Produit

S3UT-YY47-CS3P-45UK-QW4N-3YAU

Veuillez noter que cette Clé Produit ne représente qu’un exemple et qu’elle ne fonctionnera pour aucun de nos jeux !

Si vous avez téléchargé le jeu par l’intermédiaire d’un revendeur en ligne, votre Clé Produit vous aura été communiquée par e-mail ou sur l’une des pages Web qui vous ont été présentées lorsque vous avez effectué votre achat. Si vous ne retrouvez pas la Clé Produit, veuillez contacter le revendeur pour assistance.

Quand vous démarrez le jeu pour la première fois après l’avoir installé, la fenêtre d’Options d’avant jeu s’affiche vous montrant l’onglet d’Activation :

Insérez votre Clé Produit dans les six cases vides. Votre Clé Produit est imprimée au dos du manuel. Une fois la Clé Produit saisie, le bouton Sauvegarder situé sous les cases sera activé. Cliques sur le bouton Sauvegarder pour sauvegarder votre Clé Produit. L’apparence de la moitié supérieure de l’onglet d’Activation changera pour ressembler à ceci :

Si vous possédez la version DVD du jeu, vous aurez désormais le choix d’activer le jeu en ligne ou de jouer avec le disque dans le lecteur.

Jouer avec le dvd

Si vous possédez la version DVD du jeu, vous pouvez jouer sans procéder à l’activation en ligne en cliquant sur le bouton Jouer avec le DVD. Ceci n’est possible qu’une fois votre Clé Produit saisie et enregistrée. Pour que le jeu démarre, le DVD doit être présent dans le lecteur de votre Mac.

Jouer par l’activation en ligne

Si vous préférez jouer sans le DVD dans le lecteur, ou si vous avez téléchargé le jeu, vous pouvez l’activer en ligne.

Pour ce faire, cliquez sur le bouton Activer en ligne… situé sous l’onglet d’Activation.

Un dialogue apparaît (voir ci-dessous) confirmant que votre Clé Produit va être transmise à Feral. Cliquez sur le bouton Activer en ligne dans la note.

l’onglet d’Activation se remet à jour vous indiquant alors le nombre de machines sur lesquelles vous pouvez activer le jeu et sur combien de vos machines une activation du jeu est actuellement en cours de validité.

Vous pouvez désormais jouer sans disque dans le lecteur ou connexion Internet.

J’ai essayé d’activer le jeu, mais rien
ne se passe. Que dois-je faire ?

Si vous rencontrez des problèmes lors de l’activation de votre jeu, cliquez sur le bouton Contacter Feral… sous l’onglet d’Activation. Un écran de dialogue déroulant apparaîtra :

Saisissez votre adresse e-mail dans le champ Courriel ainsi qu’un résumé de votre problème dans le champ de plus grande taille, puis cliquez sur le bouton Envoyer Demande pour l’envoyer à l’équipe de support de Feral. Veuillez noter que ce formulaire ne doit être utilisé que pour les problèmes relatifs au système d’activation.

Foire aux questions

Dois-je activer le jeu pour chacun des comptes utilisateurs de mon Mac ?

Non. Une fois activé, le jeu fonctionnera pour chacun des comptes utilisateurs de votre ordinateur.

Est-ce que je peux activer le jeu à une date ultérieure ?

Oui, vous pouvez activer ou désactiver en ligne à n’importe quel moment.
